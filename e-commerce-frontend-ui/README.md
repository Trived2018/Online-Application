# 🛒 E-Commerce Web Application

## 📌 Project Description
This is a full-stack E-Commerce web application built using:

- Java (Spring Boot)
- Angular
- MySQL

The system allows users to browse products, add items to cart, and place orders. 
Admin users can manage products and orders.

---

## 🚀 Features

### User
- Register & Login
- Browse Products
- Add to Cart
- Place Orders

### Admin
- Add / Update / Delete Products
- Manage Orders

---

## 🛠️ Technologies Used
- Spring Boot
- Angular
- MySQL
- Hibernate / JPA
- Spring Security (JWT)

---

## ⚙️ How to Run

### Backend
1. Configure MySQL in application.properties
2. Run Spring Boot application

### Frontend
1. npm install
2. ng serve
3. Open http://localhost:4200
